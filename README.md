# VPN Bot для Render

Простой Telegram‑бот для продажи VPN, готовый к запуску на Render (Free план).

## Переменные окружения

- BOT_TOKEN — токен Telegram‑бота
- API_URL — URL панели VPN (опционально)
- API_KEY — ключ API панели (опционально)

## Команды

- /start — приветствие и меню
- /info — информация о VPN
- /help — помощь
- /buy — покупка (пока вручную)
